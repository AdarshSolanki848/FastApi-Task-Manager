# Task Manager using FastAPI

A simple full-stack Task Manager built with FastAPI, HTML, CSS, and JavaScript.

## Features

- Add Task
- Delete Task
- Mark Complete
- Responsive UI
- REST API
- Static file serving

## Tech Stack

- FastAPI
- HTML
- CSS
- JavaScript

## API Endpoints

GET /tasks
POST /tasks
PUT /tasks/{id}
DELETE /tasks/{id}

## Run Locally

pip install -r requirements.txt

uvicorn main:app --reload
